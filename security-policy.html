1. Security Policy Overview

This Security Policy defines the rules and procedures for protecting the network, systems, and data at Liya's Beauty Boutique. All staff and contractors must comply with these policies to ensure the confidentiality, integrity, and availability of salon systems.

2. Network Security Controls

🛡️ Firewall Rules • Block all inbound traffic by default
• Allow only required services (HTTP/HTTPS, VPN)
• Log all traffic for audit purposes
• Apply rule updates quarterly
• Monitor for suspicious activity
📶 Wireless Security • WPA3 encryption mandatory
• Separate Staff and Guest networks
• Guest network isolated from internal
• MAC filtering for staff devices
• Wi-Fi password changed quarterly
💻 Endpoint Security • Antivirus on all workstations
• Automatic OS updates enabled
• Application whitelisting where possible
• USB port restrictions
• Screen lock after 5 minutes
🔐 Access Control • Role-Based Access Control (RBAC)
• Least privilege principle
• Admin accounts restricted
• VPN for remote access
• Access reviews monthly

5. Incident Response Plan

🚨 Detection • IDS/IPS monitoring
• Log analysis weekly
• Automated alerts
• Staff reporting
📋 Response • Isolate affected systems
• Preserve evidence
• Notify management
• Document incident
🔧 Recovery • Restore from backup
• Apply security patches
• Verify system integrity
• Resume operations
📝 Post-Incident • Root cause analysis
• Update security measures
• Staff training
• Incident report
