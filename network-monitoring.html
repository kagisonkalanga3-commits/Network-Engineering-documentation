6. Alert Configuration

🚨 Critical Alerts (P1) • Device offline >5 minutes
• Security breach detected
• Backup failure
• High bandwidth usage >90%
• Complete network outage
Response: Immediate (5 min)
⚠️ Warning Alerts (P2) • High latency >20ms
• Packet loss >1%
• Low signal strength
• High CPU usage >80%
• Disk space <10%
Response: Within 1 hour
📋 Informational Alerts (P3) • Device rebooted
• Configuration changed
• Software updated
• Scheduled maintenance
• User login anomalies
Response: Logged for review
